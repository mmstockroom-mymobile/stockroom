Repair Tools
Repair existing barcodes
Cleans stale barcode suffixes (OB, MM, PO, PRE-OWNED, CPA, OBD, OBBC, etc.) from products already in the database. Also splits compound barcodes so both codes are scannable. Run this once if existing products were imported before the suffix-stripping fix. Safe to run — only changes products that need fixing, leaves units and history untouched.
